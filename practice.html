Question 1 CH02
In the simple linear regression model y = β₀ + β₁x + u, if E(u|x) = 2, then:
A. The OLS estimator β̂₁ remains unbiased
B. The OLS estimator β̂₀ is biased, but β̂₁ is unbiased
C. The OLS estimator β̂₀ will be biased
D. The OLS estimator β̂₁ will also be biased
Answer: C — The OLS estimator β̂₀ will be biased
【Explanation (EN)】
Since E(u|x) = 2 ≠ 0, the zero conditional mean assumption (MLR.4) is violated. The intercept β̂₀ absorbs this bias (biased by +2), while the slope β̂₁ remains unbiased because the error has a constant mean that does not depend on x.
【中文翻译 & 解析】
在简单线性回归模型 y = β₀ + β₁x + u 中，如果 E(u|x) = 2，则：

C. OLS估计量β̂₀将有偏。因为E(u|x)=2≠0违背了零条件均值假设，导致截距项β̂₀有偏（偏误量为2），斜率β̂₁仍无偏。
📌 考点：零条件均值假设 / Zero Conditional Mean | 章节：CH02
Question 2 CH03
In multiple regression, adding a new variable that is highly correlated (but not perfectly) with an existing independent variable typically causes:
A. R² to decrease
B. Adjusted R² to necessarily decrease
C. The standard errors of the existing coefficients to increase
D. The signs of all coefficients to change
Answer: C — The standard errors of the existing coefficients to increase
【Explanation (EN)】
High (but not perfect) correlation between regressors is called imperfect multicollinearity. OLS remains unbiased and consistent, but the variances (and thus standard errors) become large, making t-statistics smaller and coefficients appear insignificant.
【中文翻译 & 解析】
在多元回归中，加入一个与已有自变量高度相关（但非完全相关）的新变量通常会导致：

C. 已有系数的标准误增大。高度相关（多重共线性）会导致系数估计量的方差膨胀，标准误增大，但OLS仍无偏。
📌 考点：多重共线性 / Multicollinearity | 章节：CH03
Question 3 CH03
Which of the following is NOT required for the Gauss-Markov Theorem?
A. Homoskedasticity of the error term
B. Normality of the error term
C. Random sampling
D. Zero conditional mean
Answer: B — Normality of the error term
【Explanation (EN)】
The Gauss-Markov Theorem requires MLR.1–MLR.5: linearity, random sampling, no perfect collinearity, zero conditional mean, and homoskedasticity. Normality (MLR.6) is NOT required for BLUE; it is needed only for exact t/F distributions in small samples.
【中文翻译 & 解析】
以下哪一项不是高斯-马尔可夫定理所要求的？

B. 误差项的正态性。高斯-马尔可夫定理需要MLR.1-MLR.5（线性、随机抽样、无完全共线性、零条件均值、同方差），不需要正态性假设。正态性仅用于小样本t/F检验的有效性。
📌 考点：高斯-马尔可夫定理 / Gauss-Markov Theorem | 章节：CH03

Question 7 CH07

Question 8 CH08

A. The error term is normally distributed

【Explanation (EN)】

Breusch-Pagan异方差检验的原假设是：

C. 误差项是同方差的。BP检验的原假设H₀：Var(u|x₁,...,xₖ) = σ²（同方差）。备择假设：方差是解释变量的函数。

Question 9 CH08
In the presence of heteroskedasticity, using the usual OLS standard errors for t-tests leads to:
A. Always lower power
B. Incorrect Type I error probabilities (invalid inference)
C. Biased OLS estimators
D. Inability to compute t-statistics
Answer: B — Incorrect Type I error probabilities (invalid inference)
【Explanation (EN)】
Under heteroskedasticity, OLS coefficients remain unbiased and consistent (Gauss-Markov does not require homoskedasticity for unbiasedness). 【中文翻译 & 解析】

B. 第一类错误概率不正确。异方差下OLS仍无偏且一致，但标准误失效 → t和F统计量不再服从t/F分布 → 假设检验失效。
📌 考点：异方差的后果 / Consequences of Heteroskedasticity | 章节：CH08
Question 10 CH13

A. Time fixed effects

B. 个体固定效应αᵢ。一阶差分 Δyᵢ = β₁Δxᵢ + Δuᵢ，不随时间变化的αᵢ被差分消去（这正是使用面板数据的关键优势）。

Question 11 CH13
In difference-in-differences (DID) estimation, the key identifying assumption is:
A. The treatment and control groups have the same mean outcome before the policy
B. The treatment and control groups follow the same time trend before the policy

【Explanation (EN)】

在双重差分（DID）估计中，关键的识别假设是：

B. 平行趋势假设（Parallel Trends）：若无政策干预，处理组和对照组的结果变量应有相同的时间趋势。这是DID有效性的核心前提。

Question 13 CH03
When n < k+1 (sample size smaller than the number of parameters to estimate):
A. OLS can still be computed, but standard errors are invalid
B. There is perfect multicollinearity and OLS cannot be computed
C. Heteroskedasticity-robust standard errors must be used
D. R² equals 1
Answer: B — There is perfect multicollinearity and OLS cannot be computed
【Explanation (EN)】
When n < k+1, the design matrix X has fewer rows than columns of parameters, so the rank of X'X is at most n < k+1. This means (X'X)⁻¹ does not exist — a case of perfect multicollinearity — and OLS cannot be computed.
【中文翻译 & 解析】
当 n < k+1（样本量小于待估参数个数）时：

B. 存在完全多重共线性，OLS无法计算。n < k+1意味着自由度不足，设计矩阵X的秩最多为n < k+1，导致(X'X)不可逆。
📌 考点：完全共线性 / Perfect Multicollinearity | 章节：CH03

Question 15 CH04
Which statistic is specifically used to test whether several regression coefficients are jointly zero?
A. t-statistic
B. F-statistic
C. R²
D. Adjusted R²
Answer: B — F-statistic
【Explanation (EN)】
The t-test is for testing a single coefficient (H₀: βⱼ = 0). The F-test is for joint hypothesis testing: H₀: β₁ = β₂ = ... = βq = 0 (all q coefficients jointly zero). It compares restricted vs. unrestricted model fit.

📌 考点：F检验 / F-test (Joint Significance) | 章节：CH04

Question 17 CH07
For the linear probability model P(y=1|x) = β₀ + β₁x, which of the following is true?

C. 误差项必然是异方差的。因为y只能取0或1，Var(y|x)=p(x)(1-p(x))依赖于x，所以LPM的误差项天生是异方差的。预测概率可能超出[0,1]。
📌 考点：线性概率模型 / Linear Probability Model (LPM) | 章节：CH07
Question 18 CH08
When using the White test, rejection of the null hypothesis indicates:
A. Heteroskedasticity
B. Serial correlation
C. Functional form misspecification

Answer: A — Heteroskedasticity

Question 25 CH06
In the model ln(y) = β₀ + β₁x + u, the interpretation of β₁ is:
A. A one-unit change in x changes y by β₁ units
B. A one-unit change in x changes y by 100β₁%
C. A 1% change in x changes y by β₁%
D. A 1% change in x changes y by 0.01β₁ units
Answer: B — A one-unit change in x changes y by 100β₁%

Question 46 CH06
Briefly explain why adjusted R² (R̄²) is often preferred over regular R² in multiple regression.
Reference Answer (EN):
R² never decreases when a new variable is added, even if the variable is irrelevant, because it mechanically fits the sample better. This encourages overfitting. Adjusted R² imposes a penalty for the number of independent variables: R̄² = 1 − [SSR/(n−k−1)]/[SST/(n−1)]. Hence, it provides a more honest comparison across models and favors parsimonious specifications.
【Explanation (EN)】
R² measures the proportion of variation in y explained by the model. It always increases (or stays the same) when adding variables — even irrelevant ones — because OLS minimizes SSR and more variables can only reduce (or maintain) SSR. This encourages overfitting. Adjusted R² applies a degrees-of-freedom penalty: R̄² = 1 − [SSR/(n−k−1)]/[SST/(n−1)]. When adding a variable, R̄² increases only if the t-statistic on the new variable exceeds 1 in absolute value. This penalizes complexity and favors parsimonious models.
【中文题目 & 参考答案】

R²在加入新变量时永不下降（即使变量无关），因为它机械地拟合得更好，鼓励过度拟合。调整R²对解释变量个数施加惩罚：R̄² = 1 − [SSR/(n−k−1)]/[SST/(n−1)]。因此，调整R²提供了更诚实的模型间比较，倾向于简洁的模型设定。

Question 48 CH13
Compare first-difference (FD) and fixed effects (FE) estimation for panel data. What is their relationship when only two periods are available?
Reference Answer (EN):
Both FD and FE eliminate the unobserved individual fixed effect αᵢ. FD subtracts the previous period's observation from the current period; FE subtracts the individual's time-average (within transformation). For T=2 periods, the FD estimator and the FE estimator are numerically identical. For T>2, they generally differ but both are consistent under the same assumptions.
【Explanation (EN)】
Both FD and FE are methods for eliminating time-invariant unobserved heterogeneity αᵢ from panel data models. FD works by subtraction: Δyᵢₜ = yᵢₜ − yᵢ,ₜ₋₁, removing αᵢ. FE (within estimator) works by demeaning: ÿᵢₜ = yᵢₜ − ȳᵢ, also removing αᵢ. Key properties: (1) When T=2, FD and FE are numerically identical. (2) When T>2, FD is efficient if Δuᵢₜ is serially uncorrelated; FE is efficient if uᵢₜ is serially uncorrelated. (3) Both are consistent under the strict exogeneity assumption.

Question 49 CH08
What are heteroskedasticity-robust standard errors? Why are they often recommended in practice?
Reference Answer (EN):
Heteroskedasticity-robust standard errors (e.g., White standard errors) provide valid standard errors for OLS estimates even when the error variance is not constant (heteroskedasticity). They do not require knowing the form of heteroskedasticity and are asymptotically valid. Because economic data frequently exhibit heteroskedasticity, using robust standard errors avoids invalid t-tests and confidence intervals (i.e., incorrect Type I error rates).
【Explanation (EN)】
Heteroskedasticity-robust standard errors (also called White, Huber-White, or Eicker-Huber-White standard errors) are computed using the sandwich estimator: Var(β̂) = (X'X)⁻¹[Σᵢ û²ᵢ xᵢxᵢ'](X'X)⁻¹. Unlike the classical formula σ²(X'X)⁻¹, this estimator is consistent for the true variance-covariance matrix even when errors are heteroskedastic. They are 'asymptotically valid' — they work well in large samples without requiring the researcher to model the form of heteroskedasticity. This is why robust standard errors are now standard practice in applied work.
【中文题目 & 参考答案】
什么是异方差稳健标准误？为什么在实践中常被推荐使用？

异方差稳健标准误（如White标准误）即使在误差方差不恒定（异方差）的情况下也能提供有效的OLS标准误。它们不需要知道异方差的具体形式，且在大样本下有效。由于经济数据常存在异方差，使用稳健标准误可避免t检验和置信区间失效（即第一类错误率不正确）。

Question 50 CH07
List three drawbacks of using the linear probability model (LPM) with OLS.
Reference Answer (EN):
(1) Predicted probabilities can fall outside the [0,1] interval. (2) The error term is inherently heteroskedastic because Var(y|x) = p(x)(1−p(x)) depends on x. (3) The marginal effect of a variable is constant, whereas actual probabilities often change nonlinearly. (Also, standard errors must be corrected.)
【Explanation (EN)】
Three main drawbacks of the LPM: (1) Predicted probabilities ŷ = β̂₀ + β̂₁x can be negative or exceed 1, which is nonsensical for a probability model. (2) The error variance is Var(u|x) = p(x)(1−p(x)), which depends on x — so heteroskedasticity is built into the model. Robust standard errors are always needed. (3) The partial effect is constant (β₁), implying that a one-unit change in x has the same effect on P(y=1) regardless of the baseline probability, which is often unrealistic. Logistic/probit models address these issues.
【中文题目 & 参考答案】
列出使用OLS估计线性概率模型（LPM）的三个缺点。

（1）预测概率可能超出[0,1]区间。（2）误差项天生异方差，因为Var(y|x)=p(x)(1-p(x))依赖于x。（3）变量的边际效应恒定，而实际概率变化往往是非线性的。（还需使用稳健标准误进行修正。）

Question 51 CH07
Explain how to use an F-test to determine whether a qualitative variable (e.g., region with 4 categories) is jointly significant in a regression.
Reference Answer (EN):
Create g−1 = 3 dummy variables (e.g., South, North, West), leaving one category (East) as the base group. Estimate the unrestricted model including the 3 dummies and all other variables. Estimate the restricted model without the 3 dummies. Compute the F-statistic: F = [(R²_ur − R²_r)/q] / [(1−R²_ur)/(n−k−1)] with q = 3. Compare to the critical value from an F_{q,n−k−1} distribution. Rejection indicates the region dummies are jointly significant.
【Explanation (EN)】
Steps for testing joint significance of a qualitative variable with g categories: (1) Choose a base category and create g−1 dummy variables. (2) Estimate the unrestricted model including all dummies and other regressors; record SSRᵤᵣ. (3) Estimate the restricted model excluding all g−1 dummies; record SSRᵣ. (4) Compute F = [(SSRᵣ − SSRᵤᵣ)/(g−1)] / [SSRᵤᵣ/(n−k−1)]. (5) Under H₀ (all dummy coefficients = 0), this F-statistic follows an F distribution with (g−1, n−k−1) degrees of freedom. (6) Reject H₀ if F > critical value.
【中文题目 & 参考答案】
解释如何使用F检验判断一个定性变量（如4类区域）在回归中是否联合显著。

创建g-1=3个虚拟变量（如南部、北部、西部），保留一类（东部）作为基准组。估计包含3个虚拟变量和所有其他变量的无约束模型。估计不含3个虚拟变量的约束模型。计算F统计量：F = [(R²_ur − R²_r)/q] / [(1−R²_ur)/(n−k−1)]，其中q=3。与F_{q,n−k−1}临界值比较。拒绝H₀表明区域虚拟变量联合显著。

Question 52 CH13
In a difference-in-differences regression, why is it important to include both a time dummy and a treatment-group dummy? What would go wrong if one of them were omitted?
Reference Answer (EN):
The time dummy Tₜ captures common time trends affecting both groups (e.g., macroeconomic shocks). The group dummy Dᵢ captures permanent differences between treatment and control groups. Omitting Tₜ would cause the DID coefficient β₃ to absorb the common time effect, biasing the estimate. Omitting Dᵢ would allow permanent group differences to be wrongly attributed to the policy. Both are needed to isolate the causal effect of the policy.
【Explanation (EN)】
In the DID regression Yᵢₜ = β₀ + β₁Tₜ + β₂Dᵢ + β₃(Tₜ·Dᵢ) + uᵢₜ, each component plays a specific role. Tₜ (time dummy) captures common shocks affecting both groups over time — without it, β₃ would absorb the time trend. Dᵢ (group dummy) captures permanent differences between treatment and control groups — without it, pre-existing level differences would contaminate β₃. β₃ (interaction) isolates the treatment effect by netting out both common time trends and permanent group differences: DID = (ȳ_T,post − ȳ_T,pre) − (ȳ_C,post − ȳ_C,pre).
【中文题目 & 参考答案】
在双重差分回归中，为什么需要同时包含时间虚拟变量和处理组虚拟变量？如果省略其中之一会出现什么问题？

时间虚拟变量Tₜ捕捉影响两组的共同时间趋势（如宏观经济冲击）。组别虚拟变量Dᵢ捕捉处理组和对照组之间永久性的差异。省略Tₜ会使DID系数β₃吸收共同时间效应，产生偏误。省略Dᵢ会使永久性组间差异被错误地归因于政策。两者都是必要的，以分离政策的因果效应。

Question 53 CH03
Explain why including an irrelevant variable (uncorrelated with the dependent variable) generally does not cause bias but reduces efficiency.
Reference Answer (EN):
Under the classical linear model assumptions, an irrelevant variable has a population coefficient of zero. Including it does not affect the unbiasedness of the other coefficients because the zero conditional mean assumption still holds. However, it increases multicollinearity (even if only slightly), which raises the sampling variances (standard errors) of the OLS estimators, making them less precise (lower efficiency).
【Explanation (EN)】
Including an irrelevant variable (one with βⱼ = 0 in the population) does not bias the estimates of the other coefficients — OLS remains unbiased because the zero conditional mean assumption still holds (the true model is nested in the estimated model). However, including an irrelevant variable does increase the sampling variances of all coefficients because it reduces the degrees of freedom (n−k−1 gets smaller) and increases the R²ⱼ values. The consequence is larger standard errors and lower t-statistics — a loss of efficiency (precision), not a loss of unbiasedness.
【中文题目 & 参考答案】
解释为什么包含无关变量（与因变量不相关）一般不会导致偏误但会降低有效性。

在经典线性模型假设下，无关变量的总体系数为零。包含它不影响其他系数的无偏性（零条件均值假设仍然成立）。但它会增加多重共线性（即使很轻微），从而增大OLS估计量的抽样方差（标准误），使估计变得不精确（有效性降低）。

Question 54 CH07
What is the dummy variable trap? Provide an example and explain how to avoid it.
Reference Answer (EN):
The dummy variable trap occurs when a constant (intercept) is included together with dummy variables for all categories of a qualitative variable, creating perfect multicollinearity. Example: For seasons (Spring, Summer, Fall, Winter), including an intercept and four seasonal dummies makes the sum of the dummies equal to 1, which is perfectly collinear with the intercept. To avoid it, include only g−1 dummies (e.g., Spring, Summer, Fall) and treat Winter as the base category.
【Explanation (EN)】
The dummy variable trap is a perfect multicollinearity problem. Example: Suppose we have 4 seasons and include: Spring, Summer, Fall, Winter as dummies, plus an intercept. For any observation, Spring + Summer + Fall + Winter = 1, which is perfectly collinear with the intercept (a vector of ones). This violates MLR.3. The fix: drop one category (Winter) and use it as the base group. The coefficients on the remaining 3 dummies then measure the difference relative to Winter. The rule is always: g categories → g−1 dummy variables.
【中文题目 & 参考答案】
什么是虚拟变量陷阱？举例说明并解释如何避免。

虚拟变量陷阱发生在同时包含截距项和所有类别的虚拟变量时，导致完全多重共线性。例：四季（春夏秋冬），包含截距和4个季节虚拟变量时，四个虚拟变量之和恒等于1，与截距完全共线。解决方案：只包含g-1个虚拟变量（如春夏秋），以冬季为基准组。

Question 55 CH08
Outline the steps of the Breusch-Pagan test for heteroskedasticity.
Reference Answer (EN):
(a) Estimate the original regression by OLS and obtain the residuals ûᵢ. (b) Compute the squared residuals û²ᵢ. (c) Regress û²ᵢ on all explanatory variables from the original model (or a set of variables suspected to cause heteroskedasticity). (d) Obtain the R² from this auxiliary regression, say R²_aux. (e) Compute the LM statistic: LM = n×R²_aux, which under homoskedasticity follows a χ²_q distribution, where q is the number of regressors in the auxiliary regression. (f) Reject homoskedasticity if LM exceeds the critical value.
【Explanation (EN)】
Steps of the Breusch-Pagan (BP) test: (1) Estimate the original model y = Xβ + u by OLS; obtain residuals ûᵢ. (2) Compute squared residuals û²ᵢ. (3) Estimate the auxiliary regression: û²ᵢ = α₀ + α₁z₁ᵢ + … + αq zqᵢ + vᵢ, where the z's are the variables suspected to be related to heteroskedasticity (often the original x's). (4) Obtain R² from this auxiliary regression. (5) Compute LM = n · R²_aux. Under H₀ (homoskedasticity), LM ~ χ²_q. (6) Reject H₀ if LM > χ² critical value. The BP test assumes the error term is normally distributed; the Koenker (studentized) version relaxes this.
【中文题目 & 参考答案】
概述Breusch-Pagan异方差检验的步骤。

(a) 用OLS估计原回归，获得残差ûᵢ。(b) 计算残差平方û²ᵢ。(c) 将û²ᵢ对所有原模型解释变量（或怀疑导致异方差的变量）回归。(d) 获取辅助回归的R²，记作R²_aux。(e) 计算LM统计量：LM = n×R²_aux，在同方差下服从χ²_q分布，q为辅助回归中解释变量个数。(f) 若LM超过临界值，拒绝同方差原假设。
